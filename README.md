# workload management front

## 本アプリについて
workload managementはプロジェクトの工数管理を行うためのアプリケーションです。各担当者が、タスクレベルでの工数を日毎に記録できるアプリケーションです。  
このリポジトリはフロント部分です。

## 本アプリを用いて実現したいこと
- 期間(月,クォータ等)で工数を見える化し、管理,編集コストを低減させる。
  - 各人の工数を見える化する
  - プロジェクト単位で工数を見える化する
